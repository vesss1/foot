# Foot Analysis Project

This repository contains two Qt-based applications for football/soccer video analysis:

## 1. FootAnalysisGUI (Original)
A Qt application that runs Python-based YOLO object detection on videos to analyze player movements, ball tracking, and other metrics.

**Files:**
- `FootAnalysisGUI.pro` - Qt project file
- `MainWindow.h/cpp` - Main application window
- `main.cpp` - Application entry point
- `foot-Function/` - Python analysis scripts

**Purpose:** Process video files and generate analysis results.

## 2. VideoDataViewer (New)
A Qt 6.10.1 C++ application that displays analyzed video files (AVI) alongside their CSV/JSON data in a synchronized viewer with real-time playback.

**Files:**
- `VideoDataViewer.pro` - Qt project file
- `VideoDataViewer.h/cpp` - Viewer implementation
- `viewer_main.cpp` - Application entry point
- `VIEWER_README.md` - Complete documentation
- `USAGE_GUIDE.md` - Quick start guide
- `build_viewer.bat/sh` - Build scripts

**Purpose:** View and inspect the analysis results generated by FootAnalysisGUI.

## Quick Start

### FootAnalysisGUI
```bash
qmake FootAnalysisGUI.pro
make
./FootAnalysisGUI
```

### VideoDataViewer
```bash
# See USAGE_GUIDE.md for detailed instructions
./build_viewer.sh  # Linux/macOS
# or
build_viewer.bat   # Windows
```

## Workflow

```
Input Video → FootAnalysisGUI → output.avi + data files → VideoDataViewer → Visual inspection
```
