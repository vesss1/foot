## 重點說明 / Key Points

### 版本資訊顯示位置 / Version Information Display Locations

1. **視窗標題列** / Window Title Bar
   - 位置：視窗最上方
   - 格式：`Foot Analysis GUI v1.0.0`
   - 永久可見，在任務欄也能看到
   
2. **底部版本標籤** / Bottom Version Label
   - 位置：視窗右下角
   - 格式：`Version 1.0.0`
   - 顏色：灰色 (#666)
   - 字體：10pt
   - 對齊：靠右
   - 在所有標籤頁中都可見

### UI 元素說明 / UI Elements Description

1. **輸入控制區** / Input Controls
   - 影片檔案選擇 Video file selection
   - YOLO 模型選擇 Model selection
   - 開始分析按鈕 Start analysis button

2. **狀態標籤** / Status Label
   - 顯示當前狀態 Shows current status
   - 就緒/執行中/完成/錯誤 Ready/Running/Complete/Error

3. **分析日誌** / Analysis Log
   - 即時顯示 Python 輸出 Real-time Python output
   - 自動捲動 Auto-scrolling
   - 顯示進度訊息 Shows progress messages

4. **結果標籤頁** / Results Tabs
   - **摘要 Summary**: 狀態訊息 Status messages
   - **數據表 Data Table**: 球員統計 Player statistics
   - **影片輸出 Video Output**: 影片播放器 Video player

## 如何查看實際畫面 / How to See the Actual Screen

要查看實際的應用程式畫面，您需要：
To see the actual application screen, you need to:

1. **建置應用程式** / Build the application
   ```bash
   cd /path/to/foot
   qmake6 FootAnalysisGUI.pro
   make
   ```

2. **執行應用程式** / Run the application
   ```bash
   ./FootAnalysisGUI
   ```

3. **截圖** / Take a screenshot
   - Windows: 使用 Snipping Tool 或 Win+Shift+S
   - macOS: 使用 Cmd+Shift+4
   - Linux: 使用 gnome-screenshot 或 Shift+Print Screen
