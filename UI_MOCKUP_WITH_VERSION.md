## How to Change the Version

To update the application version in the future:

1. **Edit MainWindow.h**:
   ```cpp
   // Application version
   #define APP_VERSION "1.0.0"  // Change this value
   ```

2. **Rebuild the application**:
   ```bash
   qmake6 FootAnalysisGUI.pro
   make
   ```

3. The new version will appear in:
   - Window title: "Foot Analysis GUI v[NEW_VERSION]"
   - Bottom label: "Version [NEW_VERSION]"

   - **Summary Tab**: Overview and status messages
